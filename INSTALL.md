#  Guía de Instalación y Ejecución — AUTONOVA S.A.

## Descripción
Este documento explica los pasos necesarios para instalar, configurar y ejecutar el sistema **Autonova S.A.**, desarrollado con **Java Spring Boot** y **MySQL**.  
Incluye las configuraciones para el entorno de desarrollo y pruebas.

---

## 1. Requisitos Previos

Antes de iniciar la instalación, asegúrate de contar con los siguientes programas instalados en tu equipo:

| Software | Versión recomendada | Descripción |
|-----------|---------------------|-------------|
| Java JDK | 17 o superior | Lenguaje principal del proyecto |
| Spring Boot | 3.x | Framework de desarrollo backend |
| MySQL Server | 8.x | Base de datos relacional |
| MySQL Workbench | Opcional | Cliente gráfico para gestión de base de datos |
| Git | Última versión | Control de versiones |
| Maven | 3.8+ | Gestor de dependencias y compilación |
| IDE (IntelliJ / Eclipse / VS Code) | Cualquiera | Entorno de desarrollo integrado |

---
